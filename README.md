# TAREFAS

## SOBRE

#### O que é
É um aplicativo web simples que grava e deleta as tarefas.

#### Finalidade
Foi criado para prática e estudo de criação de um micro serviço com Spring Boot e de uma inteface de usuário criada com Vue.js.
